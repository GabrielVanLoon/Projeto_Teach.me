=================================================
=  SCRIPTS SQL - COMO UTILIZAR                  =
=================================================

1º  Indo diretamente no Console do PostgreSQL.
    copie e execute os scripts de `setup.sql`
    manualmente para criar o banco e o usuário.

2º  Procure e instale em seu Enviroment a biblioteca
    de interface Python <-> PostgreSQL: `psycopg2`

3º  Utilize as funções de testes criadas em `conn_test.py`
    para verificar se as conexões foram feitas corretamente.
    Se todas tiverem sido executadas corretamente então está
    tudo pronto!!!
